TODO @@@ Making the Semantic Web easy! Neologism is a simple web-based RDF Schema vocabulary editor and publishing system. Use it to create RDF classes and properties, which are needed to publish data on the Semantic Web. Its main goal is to dramatically reduce the time required to create, publish and modify vocabularies for the Semantic Web.

What Neologism is

@@@ It follows the W3C's Best Practice for publishing RDF vocabularies and Cool URIs for the Semantic Web guidelines.

What Neologism is not

  • Ontology editor
  • For large ontologies or vocabularies

Quick feature list

Limitations

  • No versioning
  • No full multi-user with several independent groups

Neologism is built on Drupal

  • Drupal 6
  • Current version not on drupal.org
  • Does not play perfectly with an existing Drupal site. We want to improve that, need help from experienced Drupalers
